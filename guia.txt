1) npm create vite@latest .
2) npm install (ejecutar npm run dev) para verificar que todo este bien

3) Instalar dependencias
    npm install react-bootstrap bootstrap
    npm install react-router-dom // para manejar las rutas
    npm install phosphor-react   // para agregar los iconos
4) importar el css de bootstrap en main.jsx antes de import './index.css'
    import "bootstrap/dist/css/bootstrap.min.css";
5) Limpiar proyecto
    borrar el contenido de App.css e Index.css
    limpiar el app.jsx 
6) Crear las carpetas datos, pages y components dentro de src
7) Colocar el archivo productos.json dentro la carpeta data
8) En componentes crear Carrusel.jsx, Footer.jsx y Header.jsx
9) En Pages Crear las carpetas home y dentro los componentes Home.jsx y Nosotros.jsx
